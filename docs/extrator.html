📋 extrator_info_files

Saiba o que há na pasta de campanha antes de abrir cada arquivo. Inventaria tipos de dado geofísico, presença de coordenadas e estágio (bruto / processado / auxiliar). Também exporta PDF, CSV e JSON e pode reorganizar o acervo em pastas padronizadas.

Status atual: v0.2.1. Processamento 100% local — lê só cabeçalhos e amostras (~2 MB/arquivo).

🔍 Inventário de campanha 📄 PDF / CSV / JSON 📁 organize 🔒 100% local

🎯 O que ela resolve

📦 A situação comum

Um HD chega com centenas de arquivos (XTF, ALL, JSF, SEG-Y, Surfer, GeoTIFF…) misturados com PDFs e fotos — e ninguém sabe o que importar primeiro.

✨ Com o extrator

Tabela colorida no terminal + relatórios opcionais. Em seguida, o comando organize copia (ou move) para pastas em inglês por método e ano.

🧰 Dois modos de uso

  • 📋 Inventário — python main.py PASTA Só leitura Classifica formatos geofísicos (HYPACK, Kongsberg, XTF, EdgeTech, Surfer, SEG-Y, GeoTIFF…). Flags: -r, --pdf, --csv, --json, --strict.
  • 📁 Organizar — python main.py organize PASTA Dry-run por padrão Saída padrão: <entrada>/Extrator_Organizer/ com bathymetry, seismic, documents, images, spreadsheets e desconhecido. Use --execute para copiar de fato.

👁️ Como fica na prática

Apontada para uma pasta, a ferramenta gera uma tabela resumida no terminal — tipos, coordenadas e status. Também é possível exportar relatórios formais.

📊 Exemplo com amostras sintéticas de demonstração.

▶️ Como começar

cd extrator_info_files
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt

# Inventário (demo do repositório)
python main.py banco_de_dados_treinamento/amostras_sinteticas -r

# Sua campanha + PDF
python main.py "D:\Campanha" -r --pdf

# Organizar (plano → Extrator_Organizer)
python main.py organize "D:\Campanha" -r
python main.py organize "D:\Campanha" -r --execute

Guia detalhado de cada comando: docs/comandos.md no repositório.
