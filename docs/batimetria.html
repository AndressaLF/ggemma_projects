🗺️ batimetria_kml_shape

Você tem profundidade + coordenadas numa planilha… e quer ver isso no mapa. Esta ferramenta lê a pasta, entende o arquivo e gera KML (Google Earth) ou shapefile (QGIS) — sem montar o mapa na mão.

Em uma frase: arquivo chato → desenho no mapa. Roda 100% no seu computador.

🌍 KML 📐 Shapefile 🖱️ Escolhe a pasta na janela 🔒 Offline

😫 O problema (sem enrolação)

📦 O que chega

Excel, CSV, TXT, XYZ, log… cada um de um jeito. Colunas sem nome claro. Às vezes UTM, às vezes graus. Abrir ponto a ponto no Excel não mostra a área.

✨ O que o programa faz

Lê a pasta, tenta achar coordenadas e profundidade, limpa pontos estranhos e entrega arquivos prontos para abrir no mapa.

🧰 Três produtos (escolha o que precisa)

  • 🌍 Conversor KML converter_batimetria2.py “Só quero ver no Google Earth hoje.” Gera .kml para conferência rápida.
  • 📐 Contornos + linhas converter_shapefile_batimetria.py · v1.8.0 “Preciso entregar para o QGIS.” Gera área (CA_…) e trajeto (L_…). Separa pedaços longe um do outro no mesmo arquivo.
  • 〰️ Só linhas converter_linhas_batimetria.py · v1.0.1 “Só o caminho da embarcação.” Exporta apenas linhas de navegação.

✨ O que você ganha na prática

🌍 Ver no Google Earth

Arquivo KML para abrir e checar se a área “cai” no lugar certo.

📐 Usar no QGIS

Shapefile de área e/ou linha, no padrão de mapa usado em muitos trabalhos no Brasil.

🧹 Limpeza automática

Tenta tirar profundidade inválida e pontos fora do esperado — menos mapa “quebrado”.

🖱️ Sem digitar caminho

Abre uma janela, você escolhe a pasta com o mouse e pronto.

▶️ Como começar (bem direto)

cd batimetria_kml_shape
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt

python converter_batimetria2.py              # KML
python converter_shapefile_batimetria.py     # área + linha
python converter_linhas_batimetria.py        # só linha

Entrada: pasta com CSV, TXT, XYZ, Excel ou LOG. Saída: na mesma pasta que você escolheu.

❓ Em 10 segundos

  • ❓ Preciso ser programador? → Só instalar Python e rodar um comando.
  • 🔒 Meus dados sobem para a nuvem? → Não. Tudo local.
  • 🗺️ Substitui o QGIS? → Não. Prepara o arquivo para você abrir lá.
